Prerequisites

- [nodejs](https://nodejs.org/en/)
- [git](https://git-scm.com/)

## Installation

Clone the project

```bash
  git clone https://github.com/renatoallart/tradingKing.git
```

Go to the project directory

```bash
  cd tradingKing
```

Install dependencies

```bash
  npm install
  #or
  yarn add
```

Start the application

```bash
  npm run dev
  #or
  yarn dev
```

```bash
  cd kingtrading
```
